## Next steps

1. Install Git if it is not already installed.
2. Open this folder in VS Code.
3. Run `git init` and `git add .`.
4. Commit with `git commit -m "Initial import of Flowchart Studio v13.5"`.
5. Create a GitHub repository and push using the commands below.

## Push to GitHub

```bash
# Replace <username> and <repo-name> accordingly
git remote add origin https://github.com/<username>/<repo-name>.git
git branch -M main
git push -u origin main
```
